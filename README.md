# FlowMate — Next.js SaaS Frontend

A modern production-ready frontend for a personal productivity SaaS app that manages tasks and habits.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- shadcn/ui-style separated components
- lucide-react icons
- Dark mode by default

## Pages

- `/` Landing page
- `/login` Login screen
- `/register` Register screen
- `/dashboard` Dashboard overview
- `/tasks` Task manager UI
- `/habits` Habit tracker UI
- `/profile` Profile/settings UI

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
